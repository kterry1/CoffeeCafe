# Coffee Cafe

## How to install and run

```js
    npm install
```

Then type:

```js
    npm start
```

### Since the Firebase authentication keys are not in the repo, the login and signup pages will not authenticate a user but otherwise the site will work as expected.
    

## Technologies Used
* React Hooks(useState, useEffect, and useContext for State Management)
* React-Router
* Framer-Motion for animation on the menu
* Firebase Authentication for user login and signup
